Django-Ash
===========
Django Admin Theme based on `Datta Able
<https://github.com/codedthemes/datta-able-bootstrap-dashboard>`_.

Requirements
------------
* Django 2.0+


Installation
------------
.. code:: bash

    pip install django-ash

Add ``ash`` to top of ``INSTALLED_APPS``:

.. code:: python

    INSTALLED_APPS = [
        'ash',
        'django.contrib.admin',
        ...
    ]
